# El Salto Limpio

Este userscript está diseñado para mejorar la experiencia de lectura en elsaltodiario.com al eliminar anuncios molestos, bloques de citas y botones de compartir.

## ¿Qué es un userscript?

Un userscript es un pequeño programa escrito en JavaScript que se ejecuta en el navegador del usuario para modificar el comportamiento o la apariencia de una página web específica. Puede ejecutarse en navegadores que soporten este tipo de scripts, como Cromite, o mediante extensiones de navegador como Violentmonkey.

## Funcionalidades del userscript:

- Elimina anuncios molestos.
- Elimina bloques de citas.
- Oculta botones de compartir.
- Añade un tiempo de lectura estimado con una variable editable.

## Notas adicionales:

Este userscript no elimina todos los anuncios, sino sólo aquellos que molestan a la lectura.
